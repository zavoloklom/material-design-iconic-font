Cheatsheet

Print this page to PDF for the complete set of vectors. Or to use on the desktop, install Material-Design-Iconic-Font.ttf, set it as the font in your application, and copy and paste the icons (not the unicode, if your switch to show it) directly from this page into your designs.

Issue with cheatsheet: If you think icons do not match the cheatsheet page, please read issue 47.
